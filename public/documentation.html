movie-api Documentation

REQUEST URL RESPONSE
GET /movies Return a list of ALL movies to the user
GET /movies/[TITLE] Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
GET /movies/[GENRE]/[TITLE] Return data about a genre (description) by name/title (e.g., “Thriller”)
GET /movies/directors/[NAME] Return data about a director (bio, birth year, death year) by name
POST /users Allow new users to register
PUT /users/[USERNAME] Allow users to update their user info (username)
PUT /users/[USERNAME]/movies/[TITLE] Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later)
DELETE /users/[USERNAME]/movies/[TITLE] Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later)
DELETE /users/[USERNAME] Allow existing users to deregister (showing only a text that a user email has been removed—more on this later)

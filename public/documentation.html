movie-api Documentation

REQUEST URL HTTP METHOD QUERY PARAMETERS REQUEST BODY DATA FORMAT RESPONSE DATA FORMAT
Return a list of ALL movies to the user /movies GET None None A JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/[TITLE] GET /movies/:TITLE None A JSON object holding data on single movie. Example:
title: "The Godfather"
description: "The Godfather is an American film series that consists of three crime films directed by Francis Ford Coppola inspired by the 1969 novel of the same name by Italian American author Mario Puzo."
genre: "Crime, Drama"
director: "Francis Ford Coppola"
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/[GENRE]/[TITLE] GET /movies/:GENRE/:TITLE None A JSON object holding the description of a genre by name/title. Example:
title: "Crime"
description: "Crime films is a film genre inspired by and analogous to the crime fiction literary genre. Films of this genre generally involve various aspects of crime and its detection."
Return data about a director (bio, birth year, death year) by name /movies/directors/[NAME] GET /movies/directors/:NAME None A JSON object holding the data of the director by name. Example:
name: "Francis Ford Coppola"
birth year: "1962"
death year: "present"

A JSON object holding the new username:
